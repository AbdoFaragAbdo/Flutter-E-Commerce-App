# e_commerce_app

Flutter E-Commerce App Using Getx StateManagement, Firebase, and MVVM

 Get-x as State Management.
 Get-x as a routing system.
 Get-x as dependency injection. 
 Firestore as backend server.
 FCM notification.
 Make a complex UI.
 Deal with real xd design.
